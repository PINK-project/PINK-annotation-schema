# The Models module of the PINK Annotation Schema
A model is a useful simplified description of a selected part of the world.

The PINK Annotation Schema categorise models as shown in the figures below.

![Top of Model branch.](https://pink-project.github.io/PINK-annotation-schema/docs/figs/models-top.svg)

Each of the three types of mathematical model is further categorised:

![Statistical model.](https://pink-project.github.io/PINK-annotation-schema/docs/figs/models-StatisticalModel.svg)
![Physics based model.](https://pink-project.github.io/PINK-annotation-schema/docs/figs/models-PhysicsBasedModel.svg)
![Data based model.](https://pink-project.github.io/PINK-annotation-schema/docs/figs/models-DataBasedModel.svg)

Within AI, natural language processsing (NLP) and machine learning is further expanded:

![Natural language processing.](https://pink-project.github.io/PINK-annotation-schema/docs/figs/models-NaturalLanguageProcessing.svg)
![Machine learning.](https://pink-project.github.io/PINK-annotation-schema/docs/figs/models-MachineLearning.svg)

Finally has machine learning the sub-branches of supervised learning and deep learning:

![Supervised learning.](https://pink-project.github.io/PINK-annotation-schema/docs/figs/models-SupervisedLearning.svg)
![Deep learning.](https://pink-project.github.io/PINK-annotation-schema/docs/figs/models-DeepLearning.svg)
