This module is a part of the [PINK Annotation Schema](https://pink-project.github.io/PINK-annotation-schema/widoco/index-en.html).A *descriptor* or *indicator* is a property that provides information about the state of a system. It simplifies complex realities into measurable values that can guide decisions.
